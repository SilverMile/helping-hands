# Helping Hands 🤝
### Connecting Seniors to Compassionate Caretakers

> Building a caring community where seniors find companionship, support, and belonging.

**🔗 Live App: [helping-hands-6ke8.onrender.com](https://helping-hands-6ke8.onrender.com)**

---

## The Problem

Dubai is home to a rapidly growing elderly population — yet finding reliable, vetted caretakers 
remains fragmented and stressful for families. Seniors often struggle to request help 
independently, and caretakers lack a structured platform to offer their services.

## The Solution

Helping Hands is a full-stack web platform that bridges the gap between senior citizens and 
caretakers in Dubai. Seniors or their families post help requests; caretakers browse, accept, 
and manage them — all in one place, with a simple 3-step flow:

1. **Request** — Tap "Request Help" and tell us what you need
2. **Match** — We connect you with a nearby, vetted helper
3. **Assist** — Help arrives: friendly, punctual, and professional

---

## Features

- 👤 **Secure Authentication** — Signup, login, and role-based access (senior / caretaker)
- 🆘 **On-demand Assistance** — Quickly request vetted helpers for errands, companionship, 
  or personal care
- ✅ **Trusted Volunteers** — Background-checked volunteers and professionals
- 📅 **Easy Scheduling** — Set recurring visits or one-time help with a few taps
- 📋 **Help Request Dashboard** — Personalised views for both seniors and caretakers
